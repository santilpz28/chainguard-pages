How I work

Methodology

Pashov Audit Group two-phase pipeline. Augmento con 12-agent walkthrough para reducir FP rate a <1%.

1. Recon & OSINT

Domain, contract, address. WHOIS, DNS, footprint, on-chain history. Establishes the threat model before any code is read.

2. On-chain check

Holders, LP lock, vesting, permissions. Etherscan API for historical verification of any operational flag.

3. Static analysis

Slither full sweep + x-ray threat model. Dedicated FP analysis: typical 89-99% FP rate filtered out.

4. Contract review

Reentrancy, approvals, invariants, access control. 12-agent walkthrough: every angle (math, gas, storage, governance) checked in parallel.

5. PoC & validation

Every finding gets a reproducer. Foundry tests for state, halmos for symbolic, manual for edge cases.

6. Report & mitigations

Markdown deliverable + 30-min readout. Severity, PoC, fix diff, and post-fix re-audit included in senior tier.
